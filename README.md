## 🎨 Detalhes do Design

### Cores Utilizadas:
- **Fundo dos botões**: `#FFFFFF` (branco)
- **Borda dos botões**: `#00C853` (verde Ton)
- **Hover - Fundo**: `#E5FFF3` (verde claro Ton)
